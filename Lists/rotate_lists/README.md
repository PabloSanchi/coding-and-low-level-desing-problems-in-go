# Rotate List

## Leetcode ref
https://leetcode.com/problems/rotate-list/

## Objective
Given the head of a linked list, rotate the list to the right by k places.

### Example
```bash 
list => 1 -> 2 -> 3 -> 4 -> 5
k = 2
```
```bash
list => 5 -> 1 -> 2 -> 3 -> 4 (first shift)
list => 4 -> 5 -> 1 -> 2 -> 3 (second shift)
```
